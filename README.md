# SmartHire-Docker

A containerized job recruitment platform built with Java Servlets, JSP, MySQL, and Docker.

SmartHire is a web-based recruitment system that connects job seekers with recruiters. The application is fully containerized using Docker for easy deployment and scalability.

### Features

- **For Candidates:**
  - Browse available job listings
  - Apply for jobs
  - Track application status
  - View application history

- **For Recruiters:**
  - Post new job openings
  - Manage job listings
  - Review applications
  - Update application status
